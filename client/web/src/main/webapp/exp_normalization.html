Expression Normalization Step

Access: Public

Overview

 The data for this study was normalized using the Supervised Normalization of Microarrays (SNM) framework. The central premise of this framework is that the study under investigation should inform the normalization process. The SNM framework consists of three components: a set of criteria that define a goal for normalization, a modeling strategy, and a set of algorithms.
