MetaGEO Project

Access: Public

Overview

Follow this project
Send leaders a message The metaGEO project was designed to build curated versions of all publicly available gene expression data. The curation process is based on a method that called Supervised Normalization of Microarrays (SNM). This method utilizes an iterative 4-step process to remove confounding variation that arises from issues related to the experimental design. All analyses undertaken within the metaGEO project utilize open source software and the resulting data sets are provided for use with no restrictions.
