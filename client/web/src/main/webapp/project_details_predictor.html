Predictive Modeling Project

Access: Public

Overview

Follow this project
Send leaders a message The ability to obtain large-scale molecular characterization of human tumors has enabled molecular diagnostic technologies designed to link the genetics of a particular tumor with predicted response to therapeutic treatments or prognostic models of clinical outcomes. The ability to develop novel predictive and prognostic models used to guide development of next-generation molecular diagnostic assays will require application of sophisticated computational methods to identify clinically relevant biomarkers by mining large-scale genomic and clinical datasets using approaches such as those developed by scientists at Sage Bionetworks.
This project represents a collaboration between Genomic Health, Inc. and Sage Bionetworks to develop novel biomarker-based predictive and prognostic models by integrating information from large-scale databases, applying and developing sophisticated computational models, and subjecting each model to rigorous performance benchmarking within reproducible workflow tools.
